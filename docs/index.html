Launch VS Code from macOS Finder with One Click

A macOS Finder service that adds "Open with VS Code" and "Open with VS Code Insiders" options to the right-click context menu for folders.

Download on GitHub Installation Guide

Features

ğŸš€ Easy Launch

Launch VS Code directly from Finder's right-click menu

âš¡ Fast Access

Open selected folders instantly without opening Terminal

ğŸ”§ Dual Version Support

Supports both VS Code and VS Code Insiders

Installation

1

Clone the repository

git clone https://github.com/neuvecom/vscode-finder-service.git
cd vscode-finder-service
2

Run the installation script

./install.sh
3

Usage

  1. Right-click on a folder in Finder
  2. Select "Services" menu
  3. Choose "Open with VS Code" or "Open with VS Code Insiders"

Troubleshooting

Services don't appear in the menu

1. Check System Preferences

System Preferences > Keyboard > Shortcuts > Services - Ensure "Open with VS Code" and "Open with VS Code Insiders" are checked.

2. Check VS Code command line tools

which code
which code-insiders

If commands are not found, create symbolic links:

# VS Code
sudo ln -sf '/Applications/Visual Studio Code.app/Contents/Resources/app/bin/code' /usr/local/bin/code

# VS Code Insiders
sudo ln -sf '/Applications/Visual Studio Code - Insiders.app/Contents/Resources/app/bin/code' /usr/local/bin/code-insiders

3. Refresh services cache

/System/Library/CoreServices/pbs -flush
killall Finder

Service doesn't work

Check logs

log stream --predicate 'process == "Finder"' --level debug

Check permissions

ls -la ~/Library/Services/

Technical Details

This service is implemented as a macOS Automator workflow:

  • Input: Folder or directory
  • Process: Execute VS Code command via shell script
  • Output: None (folder opens in VS Code)

Workflows are placed in ~/Library/Services/ and automatically appear in Finder's context menu.

Uninstallation

./uninstall.sh
